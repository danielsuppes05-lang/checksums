# Aufgaben zu Prüfsummen

In diesem Repository finden Sie verschiedene Aufgaben, die sich mit der Berechnung
und Anwendung von Prüfsummen beschäftigen.

Das Package `parity` befasst sich mit der Berechnung von Paritätsbits, einer
einfachen Methode zur Fehlererkennung in der Datenübertragung.
Die Aufgaben hier beinhalten den Umgang mit Bytes und Strings.

Das Package `digitsums` konzentriert sich auf die Berechnung von Quersummen
und deren Anwendung, z.B. zur Teilbarkeitsprüfung und zur Berechnung von
Prüfziffern nach dem EAN-13-Standard.
Bei den Grundlagen hierfür werden ähnliche Techniken wie in den Paritätsbit-Aufgaben
verwendet, jedoch muss man hier zusätzlich mit Listen arbeiten.

Beide Packages enthalten eine README-Datei, die Hintergrundinformationen und
Hinweise zur Bearbeitung der Aufgaben bietet.
